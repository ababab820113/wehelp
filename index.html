個人簡介和申請動機

我是 Abbie，一位平面設計與插畫創作者，也是一位媽媽。過去幾年，我持續在插畫與設計領域創作，喜歡透過圖像記錄生活、傳遞情感。 為了實踐能陪伴孩子成長的理想生活，也基於經濟穩定的考量，我開始思考如何用另外一種方式延伸我的創作價值。 在上一份工作結束後，我申請進入中彰投分署的全端工程師職訓課程，從零開始學習 HTML、CSS、JavaScript 等技能，逐漸對「用程式打造作品」這件事產生極大興趣。 我想申請 WeHelp，是因為我希望有一個更紮實的訓練場域，能讓我真正具備轉職為前端工程師的能力，獨立或合作的完成一套專案，也期待未來能結合創意、設計與程式，開創一條屬於我自己的全新職涯路線。

學習時間安排

孩子目前正在上幼兒園，因此在結束中彰投分署的全端工程師職訓課程之後， 我會善用平日白天（星期一到星期五）的時間全心投入學習。每天早上送孩子上學後，就會進行前端與程式相關的進修，包含撰寫課程課程專案與作業、觀看程式教學課程與紀錄筆記、查閱文件去增進我能獨立作業的能力。 晚上若有空檔，也會安排簡短的複習或觀摩他人的作品來提升實作觀念。這樣的規劃能讓我在兼顧家庭的同時，也穩定的累積實力，朝前端工程師的目標前進。

學習動機與回報期待

我本身來自設計與插畫背景，對於畫面結構與視覺呈現有一定基礎，而前端工程正好結合了設計感與邏輯思維，是我可以發揮強項、也能持續學習成長的領域。 在選擇職涯方向時，我也特別考量了工作型態的彈性。我希望能夠發展遠端工作的能力，以兼顧孩子的照顧與生活品質， 同時也相信前端工程的職能，能為我帶來更穩定且具擴展性的收入機會，朝理想的生活狀態前進。

程式開發心得

作為一位完全零基礎的學習者，兩個半月的訓練對我來說節奏非常快。 老師的教學內容深入而快速，老實說我常常跟不上，只能盡可能專注做筆記，在課堂中努力「跟著打字」。真正的理解與吸收，大多是靠著課後詢問同學、觀看線上教學影片慢慢補足的。 這份個人介紹網頁，我採用 table 切版的方式製作而不是div，原因是我在第一堂 html 課中學到的第一個練習，就是用 table 做出簡單的網頁版型。那也是我第一次透過程式語言，親手把一個畫面從結構到呈現完整的建立出來，對我來說是非常特別也難忘的經驗。 我選擇用這種方式來製作申請頁，是因為我想記住那份最初的感動。雖然整體內容仍是基礎的 html 和 CSS，距離真正的前端工程師還有一段距離，但在完成這份網頁的當下，我感受到非常強烈的成就感（以及一直來回看不知道還可以再怎麼改的小小不安）。 這次經驗讓我更堅定繼續學習的決心，也讓我開始相信：「即使起步很慢，只要願意反覆練習與修正，我也能走向理想的職涯方向。」

以工程師角度對人工智慧的看法

我認為只要是好用的工具，就應該善加利用。AI 的確是一個強大的輔助，它能幫助很多人更有效率的解決問題，對學習程式的新手來說，更像是一位隨時可以提問的助教。 但我也相信，人是複雜而多變的存在，有情緒、有直覺、有創造力。再先進的 AI，也無法完全取代人類在溝通、同理、創造與判斷上的特質。 對我來說，與 AI 合作，懂得使用它、駕馭它，才能把自己的價值發揮得更好。

想說的話

這段學習歷程對我來說，不只是轉職的準備，更是一段重新找回自信與熱情的旅程。我希望能透過 WeHelp 的訓練，打下扎實的基礎，持續精進技術、穩定成長，同時兼顧家庭與生活的平衡。我很欣賞 WeHelp 提供的「以任務驅動、實作為核心」的訓練方式，這正是我目前所需要的成長環境。誠摯期盼能有機會成為這次入選的一員，與一群願意努力的人一起踏上改變人生的起點。
